# Projet : 
Dashboard Financier Personnel
# Description
Ce projet est un tableau de bord interactif permettant de suivre mes revenus, dépenses et économies au fil du temps.  
L'objectif est d'offrir une vue claire sur ma situation financière personnelle.

# Technologies utilisées
- HTML5
- CSS3
- JavaScript
- Git & GitHub
# Nouvautée explorer 
- Manipulatin des nouveaux proprietées dans HTML5
- ajouter un boutton toggle pour changer le theme de la page 

# Difficultées rencontrées 
- Déploiment sur GitHub pages
- Ajouter un graphique interactif pour visualizer les depenses et les revenus
- Gestion du localStorage
# Solution apportées
- Graphique avec Chart.js
